# Grocery Price Comparison

A Spring Boot-based backend service for comparing grocery prices across multiple
online stores (Tesco, Auchan).  
The system supports product search, shopping list management, and full-cart
price comparison to determine which store offers the cheapest basket.

This repository contains all components of the system:
- **Backend API (Spring Boot)**
- **Frontend Web App (React)**
- **Scraper**
- **Shared documentation and architecture**
